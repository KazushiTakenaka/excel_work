# Excel & Google Sheets 自動化スターターキット

このフォルダは、AIエージェントやPythonを使ってExcel・Googleスプレッドシートを自動操作するための学習用・実務用スターターキットです。
初心者の方でも、以下の手順に沿って進めることで環境構築から基本的な操作までをマスターできます。

---

## 📂 フォルダ構成

- **01_setup_environment.md**: まずはじめにお読みください。環境構築の手順書です。
- **02_setup_service_account.md**: Googleスプレッドシートを操作するための設定手順書です。
- **03_google_sheets_basics.md**: スプレッドシート操作のサンプルコードと解説です。
- **04_excel_basics.md**: Excel操作のサンプルコードと解説です。
- **05_ai_collaboration_guide.md**: 事務職のためのAI協働ガイド（プロンプト集・自動化レシピ）です。
- **setup.bat**: ダブルクリックするだけで環境構築が完了する自動ツールです。
- **.agent/skills/**: AIエージェントに環境構築を依頼するためのスキルファイルが入っています。

---

## 🚀 使い方（おすすめの進め方）

### 1. 環境を作る
まずはPythonが動く環境を作ります。以下のどちらかの方法で行ってください。

- **方法A（一番かんたん）**:
  `setup.bat` をダブルクリックしてください。自動的に完了します。

- **方法B（エージェントに頼む）**:
  このフォルダをAIエージェントに読み込ませて、「環境構築をして」と依頼してください。

- **方法C（手動で学ぶ）**:
  `01_setup_environment.md` を読みながら、コマンドを入力して設定してください。

### 2. Googleの設定をする
- `02_setup_service_account.md` を読みながら、Google Cloudの設定を行い、`credentials.json` を取得して **`.gemini` フォルダ** に置いてください。

### 3. 動かしてみる
- `03_google_sheets_basics.md` や `04_excel_basics.md` のコードをコピペして、実際にプログラムを動かしてみましょう！

---

## ⚠️ 注意点
- **credentials.json の取り扱い**: このファイルはパスワードと同じです。他人に渡したり、インターネット上に公開しないよう十分に注意してください。
